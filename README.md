# NutriTrack

App de seguimiento nutricional semanal (React + Vite).

## Desarrollo local

```bash
npm install
npm run dev
```

## Build producción

```bash
npm run build
```

## Deploy en Vercel

Conectar el repo a Vercel. Framework detectado automáticamente: **Vite**.

- Build command: `npm run build`
- Output directory: `dist`
